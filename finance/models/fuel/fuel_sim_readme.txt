
# FUEL Advanced Simulation App

## Features:
- Dynamic agent bonding and decay
- Quantum price volatility
- Governance adjustment proposals
- Role evolution based on agent behavior

## To run locally:
1. Copy code into a React/Next.js or Vite-based app
2. Install dependencies:
   - `recharts`, `tailwindcss`, `@/components/ui` (Shadcn-compatible)
3. Add routes/components as needed
4. Start development server

## Preview:
Fully styled with neomorphic aesthetics and interactive feedback.
